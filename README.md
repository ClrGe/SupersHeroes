# LA WEB APP DES SUPERS HÉROS
## _Site web de mise en relation entre les supers héros et les municipalités_

## Prérequis

- NodeJS ^16.14.0
- NPM ^8.1.0
---
## Installation

```bash
git clone git@github.com:ClrGe/SupersHeroes.git
cd SupersHeroes
npm install
```
---
## Stack technique

- SvelteJS
- TailwindCSS
- MapBox
